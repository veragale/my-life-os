# 效率工具的陷阱

## 工具收集癖

我曾经是一个疯狂的效率工具收集者：

- Notion、Obsidian、Logseq、Roam Research 全都试过
- 各种 GTD 方法论：番茄钟、时间块、四象限...
- Chrome 插件装了 50+，却很少真正用

## 顿悟时刻

直到某天我意识到：**我在"优化工作流"上花的时间，已经超过了真正做事的时间。**

工具不是解药，行动才是。

> 最好的生产力系统，是让你忘记系统本身的存在。

---

**当前实践**：
- 只用 3 个核心工具（VS Code + Terminal + Browser）
- 纸笔 + Markdown 文件管理所有想法
- 减少选择，增加执行
