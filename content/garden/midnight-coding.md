# 深夜编码的诗意

## 凌晨两点的键盘声

有人说深夜写代码是效率低下的表现，但我知道，那是另一种维度的清醒。

当世界安静下来，代码变得纯粹：

```javascript
// 没有会议，没有打断
// 只有你、编辑器、和那个待解决的问题
const flow = silence => focus => creation;
```

## 为什么深夜适合写代码？

- **认知连续性**：思维不会被打断，可以在脑中完整构建复杂的抽象
- **感官纯粹**：夜晚屏蔽了白天的噪音，让人专注于逻辑本身
- **时间的错觉**：深夜的一小时，感觉像白天的三小时

> 代码是理性的艺术，深夜是感性的容器。两者结合，诞生最好的作品。

---

**深夜编码守则**：
1. 备好咖啡或茶
2. 放一张环境音乐（Lo-fi / Post-rock）
3. 关掉所有通知
4. 第二天早上 review 代码（深夜的 bug 会在白天现形）

**推荐音乐**：
- Nils Frahm - Says
- Ólafur Arnalds - Near Light
- Max Richter - On the Nature of Daylight
